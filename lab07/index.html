Ejercicio

Actividad

Tomando como base el procedimiento para generar consultas que permitan operaciones de bases de datos (CRUD) MySqli con PHP visualizado la semana pasada, realizar una aplicación que resuelva lo que se plantea a continuación. Se desea implementar una biblioteca digital que permita a los estudiantes de TECSUP acceder a bibliografía por medio de internet. El funcionamiento será el de un catálogo de libros en el cual el estudiante pueda seleccionar el libro que desea leer.

Libros

Lista de Libros

Un libro debe considerar dentro de sus atributos: id, año, autor, titulo, URL de ubicación del recurso, especialidad y editorial.
Se debe permitir:

  • Mostrar la lista de libros
  • Agregar un nuevo libro
  • Editar la información de un libro
  • Eliminar un libro
